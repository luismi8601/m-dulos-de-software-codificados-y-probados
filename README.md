# Sistema de Inventario de Pijamas

Proyecto web desarrollado con HTML, Bootstrap y JavaScript que permite gestionar un inventario de pijamas.

## Funcionalidades

- Agregar productos
- Modificar productos
- Eliminar productos
- Buscar pijamas
- Validaciones de datos

## Cómo usar

1. Descargar el proyecto
2. Abrir el archivo inventario.html
